# UDP Connect Sender

![](../UNO_R4_WiFi_Connect_Empfaenger/demo_connect.avif)

Für dieses Beispiel brauchst du zwei Arduino UNO R4 WiFi.
Einer wird der `Sender`, der zweite der `Empfänger`.

Es gibt jeweils einen passenden Beispiel-Sketch dafür.

- Schließe einen Taster an D2 (z. B. über das Grove-Shield) an.
- Lade den Sketch hoch.
- Wenn du den Taster drückst, sendet der Arduino diese Information an den anderen Arduino.

## Aufgabe

- Tausche den Taster gegen einen Dreh-Regler aus.
- Passe dann den Sketch so an, dass der Analogwert gelesen und versendet wird.
