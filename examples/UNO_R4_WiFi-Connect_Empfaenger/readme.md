# UDP Connect Empfänger

![](./demo_connect.avif)

Für dieses Beispiel brauchst du zwei Arduino UNO R4 WiFi.
Einer wird der `Sender`, der zweite der `Empfänger`.

Es gibt jeweils einen passenden Beispiel-Sketch dafür.

> [!info]
> Sketch für Empfänger

Du benötigst keine externen Bauteile.

- Lade den Sketch hoch.
- Wenn der Sender eine Info sendet, wird die On-Board-LED entsprechend ein- oder ausgeschaltet.

## Aufgabe

Passe den Sketch so an, dass ein empfangener Analogwert als Helligkeit auf der LED angezeigt wird.
