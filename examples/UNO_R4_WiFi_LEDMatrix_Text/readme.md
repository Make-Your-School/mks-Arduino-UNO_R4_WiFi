# UNO R4 WiFi - LEDMatrix - scrolling Text

du benötigst keine externen Bauteile.

lade einfach den Sketch hoch.
dann sollte die LED Matrix den Text scrollend anzeigen wie in diesem Video:
![text animation](<text upload-scaled-600.avif>)

## Aufgabe

### änder den Text ab

Es gibt im Sketch zwei Stellen wo der Text definniert ist.
eine im `setup` und eine in der `loop`.
probiere was sich wie ändert.
